# Phase 8 Context: Guided Workflow and Completion Signals

## Why This Phase Exists

BIOMIX v1.0 already computes the right things, but first-time users can still hesitate because the product does not yet show enough progress, completeness, and readiness guidance in one place. The next milestone should reduce that hesitation before adding deeper features.

## User Pain We Are Solving

- Users do not always know where to begin after landing on the dashboard.
- Users can edit forms, but they cannot yet see a simple completion view for the whole scenario.
- Users can see validation and feasibility outputs, but not one compact readiness summary that combines both.

## Phase Scope

- Add stronger workflow guidance from dashboard into the input flow
- Add per-section completion tracking across the main input groups
- Add a compact readiness summary for the active scenario

## Out of Scope

- No new calculation formulas
- No backend/auth work
- No major redesign of analysis or persistence flows

## Acceptance Shape

This phase is done when a user can land on the app, see the recommended next step, understand which input groups remain incomplete, and judge whether the active scenario is ready for review.
