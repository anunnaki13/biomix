# Plan 08-02: Input Completion Tracking

## Goal

Help users understand which input groups are finished and which still need work.

## Tasks

1. Define completion checks for production, feedstock, quality, pricing, OPEX, CAPEX, transport, and financing
2. Surface completion indicators in the input flow and/or related navigation
3. Highlight missing groups without blocking normal editing

## Verification

- Completion state updates when the active scenario changes
- Users can identify unfinished sections quickly
- Completion indicators do not conflict with schema validation
